# -# 车来了智能出行助手

## 项目简介
基于扣子（Coze）平台构建的AI出行Agent，解决公交路线查询、实时追踪、末班车查询等场景问题。

## 核心功能
- 路线规划：起点到终点公交路线查询
- 实时公交：指定线路到站时间查询
- 末班车查询：线路末班车时间
- 常见问题解答：车来了APP使用问题
- 投诉反馈：生成工单

## 技术实现
- 平台：扣子（Coze）
- 工作流 + 大模型 + 知识库（RAG）
- 分支路由、代码节点、API调用
- 知识库覆盖25+常见问题
